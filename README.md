# Проект tiitt10_4

## Описание

Это учебный проект, предназначенный для отработки навыков работы с Git и GitHub. Проект содержит простую программу на C++, которая вычисляет минимальное, максимальное и среднее значение в массиве.

## Функциональность

- Вычисление минимального значения в массиве
- Вычисление максимального значения в массиве
- Вычисление среднего арифметического значений массива

## Использование

### Требования

- Компилятор C++ (поддерживающий C++11 или выше)
- Система сборки CMake (опционально)
